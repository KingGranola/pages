01

Getting started

Getting started

How to read this manual

The same basic workflow applies to iPhone and iPad. Blue numbers on screenshots match the numbered explanations beside or below each image. App control labels are shown in English, such as LOAD SAMPLE.

What TXT sampler does

SAMPLE

Prepare sounds

Record from the microphone or import an audio file, then shape the waveform and sound.

SEQUENCE

Build rhythms

Place sounds on a 16-step grid and play them back as a loop.

PROJECT

Keep your work

Save samples, pad settings, sequences, and BPM, then export finished loops as WAV audio.

Play your first sound

  1. 1

    Open SAMPLE and choose a pad

    Tap the pad that should receive the sound. The selected pad name appears in the waveform area.

  2. 2

    Tap LOAD SAMPLE

    Choose an audio file. It will be loaded into the currently selected pad.

  3. 3

    Tap the pad to check the sound

    If the sound plays after loading, the pad is ready.

03

SAMPLE mode

SAMPLE mode
Record and shape samples

  1. 1
    Waveform display

    Check the sample name and waveform, then set the playback range with START and END.

  2. 2
    Sample actions

    Use REC, ADSR, LOAD SAMPLE, DEL SAMPLE, and RESET.

  3. 3
    Pad selection

    Select the pad you want to edit first.

  4. 4
    CTRL tab

    Adjust pitch, filter, volume, and playback behavior.

Import an audio file

  1. Select the destination pad.
  2. Tap LOAD SAMPLE.
  3. Choose an audio file and confirm that the waveform appears.
Import duration limit

The free plan can import audio up to 10 seconds. The Premium plan can import audio up to 60 seconds.

Record with the microphone

  1. Select the destination pad and tap REC.
  2. Allow microphone access the first time you record.
  3. Stop recording, then choose “Use this recording” on the confirmation screen.

If the destination pad already has a sound, confirming the recording replaces the existing sample.

Trim the waveform

Move START and END on the waveform to show TRIM. Running TRIM overwrites the sample with the current range.

TRIM cannot be undone

Duplicate the project first if you need a backup. Pad settings such as pitch and ADSR are kept.

Shape the sound

PITCHChange pitch in semitone steps
CUTOFFReduce high frequencies
RESONANCEEmphasize the cutoff area
AMPAdjust volume per pad
ADSRAdjust attack, decay, sustain, and release
REVERSEPlay the sample backward
NORMALIZEAutomatically adjust sample level
MUTEMute the selected pad

Change playback behavior

GATE
Plays only while the pad is held.
LOOP
Repeats the selected playback range.
ROLL
Repeats the sound at the selected roll rate.
CHOKE
Stops sounds in the same group when another sound in that group plays.

04

SEQUENCE mode

SEQUENCE mode
Build patterns

  1. 1
    16-step grid

    Tap cells to turn steps on or off.

  2. 2
    Bar and Swing controls

    Adjust bar count, Swing, and the selected pad pattern. Use the arrows on the left and right of the grid to move between bars.

  3. 3
    Pad selection

    Changing pads switches the grid to that pad’s sequence.

  4. 4
    TRANSPORT

    Control playback, stop, realtime recording, and quantize.

Step input

  1. Select the pad you want to edit.
  2. Tap the 16-step grid to turn on positions where the sound should play.
  3. Press play to check the loop.
  4. Select another pad and repeat the same process to add more sounds.

Adjust timing

BPM

Tempo

Changes the overall speed.

SWING

Groove

Delays even-feel offbeat steps for a swung rhythm.

REC

Realtime recording

Records pads you hit while the sequence is playing.

QTZ

Quantize

Aligns realtime-recorded hits to the step grid.

Add bars

BAR+ adds a bar, and BAR- removes the last bar. The free plan is limited to 2 Bars. The Premium plan supports up to 8 Bars.

SEQUENCE effects

In SEQUENCE mode, the four knobs work as DIST., DL. TIME, DL. FBK., and DL. LV.

CLEAR clears the sequence for the selected pad.

Import and export sequence JSON

Tap IMP EXP to choose IMPORT JSON or EXPORT JSON. You can import or export the sequence BPM, Swing, step data, and SAMPLE parameters as JSON.

IMPORT JSON overwrites the current project sequence and SAMPLE parameters. Audio files are not included.

PROJECT mode

PROJECT mode
Save and export audio

  1. 1
    Project information

    Shows sample count, missing files, and data size.

  2. 2
    Management buttons

    Use NEW, COPY, DELETE, and EXPORT WAV.

  3. 3
    Pads

    Pressing a pad in PROJECT moves to SAMPLE and plays that sound.

  4. 4
    TRANSPORT

    Play or stop the saved pattern.

Manage projects

A project stores samples, pad settings, sequences, BPM, and related work data.

NEW

Create an empty project.

COPY

Duplicate the current project. Use it as a backup before editing.

DELETE

Delete the selected project.

EXPORT WAV

Export the current sequence as a WAV file.

DELETE cannot be undone

Check the target project carefully before deleting from the list menu or swipe action.

Export WAV

  1. Tap EXPORT WAV.
  2. Choose whether to include effects.
  3. Choose the loop count from 1 to 32.
  4. Tap “Export WAV,” then choose a share or save destination.
Effect tail

When effects are included, a 2-second tail is added to preserve reverb or delay decay. Without effects, the WAV is exported at the exact sequence length.

WAV export is a Premium feature.

08

FAQ

FAQ

No sound plays
  1. Check that a waveform is visible on the pad.
  2. Confirm that MUTE is off.
  3. Raise AMP and the device volume.
  4. Check the output destination for Bluetooth or external audio devices.
I cannot import a sample

Select the destination pad first, then tap LOAD SAMPLE. Check that the file length is within the plan limit.

I cannot record with the microphone

Allow microphone access for TXT sampler in iOS Settings. Then choose the destination pad and tap REC.

A project waveform is missing

Select the target pad and load the audio again from LOAD SAMPLE.

Purchased features are unavailable

Connect to the network and run Settings > About > Restore Purchases. Make sure you are using the same Apple Account used for purchase.

The screen locks automatically

Turn on “Keep screen awake” in Settings > Performance Settings.

I want to reset the app

Use FACTORY RESET on the rear panel. This deletes all projects, samples, sequences, and app settings, and cannot be undone.
